# UI Reference Notes

The first visual direction follows mature marketplace patterns without copying a single source.

- GitHub Marketplace: strong search, clear categories, recommended and recent listings, developer-oriented navigation.
- Product Hunt: ranked product launches, compact tag metadata, social proof, and high-density discovery.
- Setapp: curated app catalog, trusted selection, simple onboarding, and clear developer entry points.

The Figma community search URL was noted as a design reference, but it was not fetchable from this environment. The current screen uses the proposal requirements plus the marketplace references above.

Reference URLs:

- https://github.com/marketplace
- https://www.producthunt.com/
- https://setapp.com/
- https://www.figma.com/community/search?query=App%20Store&resource_type=files&editor_type=all&price=all&sort_by=relevancy&creators=all
